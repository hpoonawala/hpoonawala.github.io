  • Authors: Driess, Danny; Schmitt, Syn; Toussaint, Marc

Broad area/overview

This paper proposes an inverse model for a redudant system which has applications in learning to control musculoskeletal systems. The contributions listed are:

  1. Formalize the meaning of 'learning an inverse model'.

  2. Propose a method to find the reachable workspace.

  3. In order to improve the inverse model, the training model of the forward model is efficiently generated.

  4. To get a guaranteed uper bound on the performance in real systems.

Notation

  • \(\hat{\phi}\): True function.

  • \(<.,.>_{H_{k}}\): ineer product of the Reproducing kernel Hilbert Space (RKHS) associated with k.

  • \(\chi\): Reachable set.

  • \(\pi\): Inverse Model

Specific problem

Owing to the disadvantages of sparsity of the data used while using just the forward model for training, the paper approaches to estimate the the reachable set using learned models.

Solution ideas

  • Coining a function using kernel based methods to keep in check the error that is obtained between the true and the learned functions.

  • Kernel ridge expression:

    • \(\min_{\phi \in H_{k}}\sum_{i=1}^n\mid x_{i}-\phi(u_{i})\mid^{2} + \sigma^{2}\parallel\phi\parallel^{2}_{H_{k}}\)

    where \(\sigma \geq 0\) is the regularization parameter.

  • Error estimate:

    • \(\mid \hat{\phi}(u)-\phi(u)\mid \leq \parallel\hat{\phi}\parallel_{H_{k}} s(u)\).

  • The forward mdoel is first learned using the control inputs and the corresponsing configurations obtained through these inputs.

  • Given this model, the true reaching error is calculated.

  • For estimating the workspace, error certaininty is used to define the true reachable set defined by the error certaininty \((c)\), Th definition:

    • \(\hat{\chi}_{c} =\left\{x\in\mathbb{R}^{d}: dist(x,\chi)

  • The paper uses fill-distance for the exploration strategy. For an efficient data collection, the entire space of the samples that constitute the unknown workspace \(\chi\) is selected fot he exploration strategy.

  • To validate the results, following experiments have been performed, 150 data points have been used that completes the reachable set. It was seen that the estimated and the reachable true set match the true-fill distance.

Comments

  • The results have been obtained from an actual musculoskeletal system and so have also been tested for a real-world scenario.

  • This paper can provide good insights to people who are working on 3D origami robotics.
